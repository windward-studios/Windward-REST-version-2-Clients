Windward REST .Net Standard 2.0 Client

Windward Studios is a fast, easy, and powerful system that allows you to quickly layout reports using Microsoft Word, Excel, or PowerPoint and then create documents in numerous output formats format using a SQL database, JSON file, XML file, or OData for the data and the Word, Excel, or PowerPoint document you created as the report template.

How it Works

Very simply, with Windward you lay out your report in Microsoft Word, Excel, or PowerPoint. In Word/XL/PPT you use all of it's powerful layout and formatting properties to set exactly how you want the document to look.

Once you have created the template, it is saved. You then feed the Windward RESTful engine this template file and one or more datasource(s), and it then merges the data into the template and creates a final document, with all of the formatting and positioning as set in your original template. This final document can be written in numerous standard formats.

Calling the API

The following is simple to show the API calls. You normally will not want to call in this way as it makes the entire set of code synchronous. Using await/async you can call all this in an asynchronous manner.

 

// we have nothing else to do, so we wait till we get the result.
Document document = client.PostDocument(template).Result;
string guid = document.Guid;

// wait for it to complete
// instead of this you can use: template.Callback = "http://localhost/alldone/{guid}";
while (client.GetDocumentStatus(guid).Result != HttpStatusCode.Found)
    Thread.Sleep(100);

// we have nothing else to do, so we wait till we get the result.
document = client.GetDocument(guid).Result;

// delete it off the server
client.DeleteDocument(guid);
